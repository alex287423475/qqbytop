# 商务形象诊断报告结构

## 免费版

免费版用于验证转化，展示内容控制在 30 秒内能读完：

- 信任感评分
- 当前主要问题
- 推荐商务形象方向
- 3 条立即可执行建议

## 付费版 12 项

1. 头像构图与裁切
2. 表情与视线
3. 服装色彩
4. 背景与光线
5. 姿态与肩颈线条
6. 跨文化信任感
7. 行业身份匹配
8. LinkedIn 首屏表达
9. 英文个人简介
10. 官网 About 页面
11. 展会/名片场景
12. 30 天执行清单

## 付费分档

- 免费诊断：评分、主要问题、3 条建议。
- 标准报告：完整 12 项报告 + 2 张商务形象参考图。
- 专业形象包：完整 12 项报告 + 6 张参考图 + LinkedIn/官网英文简介。
- 定制海外商务形象设计方案：AI 报告 + 人工复核 + 海外商务形象设计建议 + 英文个人品牌文案。

参考图的定位是“拍摄参考 / 风格方向图”，不承诺替代真实职业照，也不做面相、命运、健康或敏感属性判断。

## AI 输出 JSON 草案

```json
{
  "score": 76,
  "issue": "背景与服装信息不足",
  "direction": "清爽可信的顾问型形象",
  "summary": "适合先从头像背景、服装颜色和英文简介定位三处改起。",
  "quickInsights": ["...", "...", "..."],
  "paidModules": ["头像构图与裁切", "表情与视线"]
}
```

正式接入 AI 时，后端必须校验 JSON 字段，不能直接把模型输出透传给前端。
