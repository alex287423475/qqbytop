# 雅思口语诊断工具 SEO / 小红书 / 题库流量方案

更新日期：2026-05-05

## 1. 结论

> 本方案已调整为主产品 `/tools/ielts-speaking` 的流量附件。早期“纯文本轻量诊断页”思路不再作为独立 V1 执行；文本诊断只作为题库页的辅助入口，核心产品仍以“录音 + 腾讯云 SOE + 基础报告/深度报告”为准。

建议做，但不要把它定位成“AI 雅思打分器”。更稳的定位是：

> 免费雅思口语题库作答诊断工具：基于当季题库、录音作答、腾讯云 SOE 发音评测和公开评分维度，输出口语弱项、提分优先级和可复用表达建议。

它适合作为留学服务入口，而不是独立卖课产品。对全球博译更自然的后链路是：

- 留学文书英文润色、PS / SOP 表达优化。
- 英文简历、推荐信、邮件、面试表达修改。
- 成绩单、在读证明、毕业证、奖学金材料等翻译。
- 留学申请材料包语言一致性审校。

## 2. 为什么这个方向有流量

### 2.1 题库有周期性搜索需求

雅思口语题目按考试窗口滚动，当前日期是 2026-05-05，正处在 2026 年 5-8 月题库窗口。换题季带来的关键词包括：

- 2026 年 5-8 月雅思口语题库
- 雅思口语 Part 1 题库
- 雅思口语 Part 2 cue card
- 雅思口语当季新题
- 雅思口语换题季怎么准备
- 雅思口语题库答案

这些词天然适合做“题库页 + 作答诊断”组合，而不是只发文章。

### 2.2 官方评分维度稳定，适合做诊断报告

IELTS 官方口语评分维度稳定为四类：

- Fluency and coherence
- Lexical resource
- Grammatical range and accuracy
- Pronunciation

工具应该明确声明：诊断结果不是官方分数，只是基于公开评分维度的练习反馈。

### 2.3 小红书适合承接焦虑型搜索

小红书用户的搜索意图不是“读完整攻略”，而是：

- 我现在这段回答能几分？
- 5-8 月口语新题怎么准备？
- Part 2 卡住怎么办？
- 为什么我口语一直 5.5 / 6？
- 有没有一句话模板能救急？

因此小红书不要直接推“翻译公司”，而要做“题库 + 诊断 + 反馈截图”的内容钩子。

## 3. MVP 产品形态

### 3.1 页面路径

建议路径：

```text
/tools/ielts-speaking
```

中文 SEO 标题：

```text
雅思口语诊断工具｜2026 当季题库作答检查与提分建议
```

描述：

```text
选择 Part 1 / Part 2 / Part 3 题目，录音回答或补充粘贴英文文本，免费获得流利度、词汇、语法和发音表达风险诊断，适合 2026 年 5-8 月雅思口语题库练习。
```

### 3.2 输入

V1 主入口做录音诊断，文本输入只作为 SEO 题库页的辅助降级入口。若用户无法授权麦克风，可以粘贴英文回答获得不含真实音素证据的文本诊断。

字段：

- 考试月份窗口：2026 年 5-8 月、2026 年 9-12 月、其他。
- 题型：Part 1 / Part 2 / Part 3。
- 题目主题：旅行、天气、运动、购物、音乐、建筑、环境、帮助他人、书/电影、目标等。
- 目标分数：5.5 / 6 / 6.5 / 7 / 7+。
- 作答音频：用户使用浏览器录音提交。
- 作答文本：可选。用于用户无法录音时的降级诊断，或用于人工修正转写。
- 最担心的问题：卡顿、词汇简单、语法错、回答太短、没有例子、Part 3 没观点。

### 3.3 输出

报告结构：

- 综合风险：低 / 中 / 高。
- 预计区间：例如“更接近 5.5-6.0 练习表现”，避免承诺真实考试分数。
- 四维诊断：
  - 流利度与连贯：回答长度、逻辑连接、重复表达。
  - 词汇资源：主题词、同义替换、搭配自然度。
  - 语法范围与准确性：句型复杂度、错误是否影响理解。
  - 发音表达风险：录音模式基于 SOE 音素/单词证据；文本降级模式只提示表达风险，不做真实音素判断。
- 当题提分建议：3 条可执行修改。
- 一段示例改写：只改 80-120 词，避免替用户整篇代写。
- 下一步 CTA：领取完整 7 天口语表达修正表 / 咨询留学材料英文润色。

## 4. SEO 内容矩阵

### 4.1 工具主词

优先做低竞争长尾：

- 雅思口语诊断工具
- 雅思口语评分工具
- 雅思口语回答检查
- 雅思口语 AI 诊断
- 雅思口语 Part 2 回答修改
- 雅思口语 6 分到 6.5

### 4.2 题库页

题库页用于吃搜索，工具用于转化。

页面结构：

```text
/ielts-speaking/2026-may-august/
/ielts-speaking/part-1/travel/
/ielts-speaking/part-1/weather/
/ielts-speaking/part-2/memorable-journey/
/ielts-speaking/part-2/environmental-problem/
/ielts-speaking/part-3/technology/
```

每个页面必须有：

- 当前题目窗口说明。
- 题目分类。
- 5.5 / 6.5 / 7 分回答差异。
- 常见中文思维错误。
- “录音回答这道题”按钮。
- “无法录音？粘贴英文文本做简版诊断”辅助入口。

### 4.3 问题型文章

适合站内博客和 AI SEO：

- 雅思口语 Part 2 说不满 2 分钟怎么办？
- 雅思口语 5.5 到 6.5，最先改哪三个问题？
- 雅思口语词汇不是越高级越好吗？
- 雅思口语 Part 3 没观点怎么办？
- 雅思口语答案背模板会扣分吗？

## 5. 小红书内容打法

### 5.1 栏目

每周固定 4 类：

1. 当季题库卡片：一题一页，附“别这样答 / 可以这样扩展”。
2. 诊断截图：匿名展示用户回答问题和系统反馈。
3. 分数差异：同一题 5.5、6.5、7 的回答差别。
4. 留学材料联动：口语表达差，也会影响面试、PS、邮件和简历表达。

### 5.2 标题模板

- 5-8 月雅思口语新题，这样答很容易 5.5
- Part 2 说不满 2 分钟？先改这个结构
- 你的雅思口语不是词汇少，是例子太空
- 雅思口语 6 到 6.5，别再背万能句了
- 我把一段 Part 2 回答诊断了一遍

### 5.3 笔记结尾 CTA

不要写硬广，建议用：

```text
想测自己的回答，可以在诊断工具里录音回答一道题，先看四个维度哪里最拖分。
```

标签：

```text
#雅思口语 #雅思口语题库 #雅思换题季 #雅思备考 #留学申请 #英文表达 #雅思口语Part2
```

## 6. 线索分层

### 6.1 免费用户

只给：

- 题目诊断。
- 三条修改建议。
- 一段短示例。

### 6.2 邮箱 / 微信线索

给：

- 完整 PDF 报告。
- 7 天练习清单。
- 同主题 10 个替换表达。

### 6.3 付费服务入口

根据报告分流：

- 口语表达弱：英文表达修正 / 面试表达训练。
- 语法和词汇弱：英文润色包。
- 留学申请中：PS / SOP / CV / 推荐信语言优化。
- 材料准备中：成绩单、证明、证书翻译。

## 7. 技术落地顺序

第一阶段，1-2 天：

- 完善 `/tools/ielts-speaking` 工具页。
- 题库入口直接连接录音诊断。
- 文本输入作为降级入口，不作为主产品路径。
- 加入 sitemap。
- 用 demo fallback，后端未接入时也能跑。

第二阶段，3-5 天：

- 增加题库主题页。
- 诊断报告可分享。
- 线索写入本地 JSON / 后台查看。
- 增加 Search Console 关键词追踪。

第三阶段：

- 强化录音上传、SOE 结果和转写校正。
- 加小红书素材自动生成。
- 按题库窗口批量生成内容页。

## 8. 风险

- 不要宣称“官方评分”或“保证提分”。
- 不要复制受版权保护的完整题库内容；使用题目主题、练习型改写和用户自填问题。
- 发小红书时不要伪装成真实考官评分。
- 诊断页必须写清楚隐私和数据使用边界。

## 9. 当前可执行的下一步

优先级最高的是做一个轻量工具页，而不是先批量铺题库页。原因是：

- 工具页能验证转化。
- 小红书可以立刻导流。
- 后续题库页都可以挂同一个诊断入口。

建议先开发：

```text
/tools/ielts-speaking
```

配套第一批 10 个题库主题页和 20 条小红书标题即可开始测试。
